# vedetta
*Open*BSD Router Boilerplate

## About
> an opinionated, best practice, vanilla OpenBSD base configuration for bare-metal, or cloud routers

## Features
* acme-client
* authpf
* dhclient
* dhcpd
* (dhcp6s and dhcp6c optional, from package)
* ftp-proxy
* httpd
* ifstated
* iked
* ipsec
* isakmpd
* npppd
* nsd
* pf
* rebound
* relayd
* rtadvd
* sshd
* tmux
* unbound
